Github Copilot 初步使用方法

代码补全

这是最简单的一个功能，无需任何操作，只是在编写代码的时候，Copilot会默默地提供建议，按alt键接受建议。

根据注释生成代码

只需要在编辑器中打出注释# 验证邮箱地址，后续会逐句生成一个函数。

注意：按下tab表示接受这一步的建议，然后按下enter，会生成下一步建议。

也可以按下alt+]表示切换下一个建议；

按下alt+[表示切换上一个建议；

按下enter表示接受这一步的建议；

按下esc表示拒绝此次建议。

也可以点击右边的Copilot按钮，直接查看所有建议，点击Accept solution，插入此项建议。

编辑器内问答

在编辑器内打出# q: 表示要提出一个问题，后续按下enter，会生成一个回答，以# a: 开头。按下tab表示接受这个回答，再按下tab，可以进行下一次提问，支持连续对话。

关于Github教育优惠

成功通过GitHub的学生认证，主要步骤如下：

  1. 学信网下载表弟的学籍验证报告。学信档案
  2. 把学籍验证报告中的部分文字改成英文，比如姓名、学校、就读方式、日期之类的。
  3. GitHub认证页面填写信息，提交即可。Request a discount - GitHub Education

需要注意的点：

  1. 不要开科学上网，最好在学籍中的大学所在城市申请。（我在异地申请也通过了）
  2. Adobe Acrobat DC 修改PDF文件很方便。Acrobat DC破解版
  3. Github账号的 Settings / Public profile 中的姓名，个人介绍，城市之类的要与学籍信息高度一致。
  4. Github账号的 Settings / Billing & plans / Payment information / Billing information 里的信息要与学籍信息高度一致。
  5. 关于邮箱，如果没有特殊要求，可以使用非教育邮箱。（吐槽CCZU，从未提供教育邮箱，Github却又要求提供）
  6. 上传学籍验证报告的时候最好是拍照上传，不要直接上传文件。

扩容分区和文件系统

查看磁盘是否有剩余空间

可以看出本来磁盘有200个G，挂载到/data目录下，现在增加了200G容量，/dev/vdc容量有400G，但是新增加的200G还处于未分配状态。

分区扩容

使用grow_part命令，

# 安装软件包
yum install cloud-utils-growpart
yum install xfsprogs

# 分区扩容，grow_part 分区 分区号
grow_part /dev/vdc 1

扩容文件系统

xfs文件系统使用xfs_growfs命令，其他文件系统使用resize2fs命令

xfs_growfs /dev/vdc1

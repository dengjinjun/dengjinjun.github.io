    sed

    文本处理三剑客——流式编辑器sed

    基本用法：

    sed [-options] '规则' [文件]

    sed会将文本内容逐行读取到内存中，利用规则处理好之后输出到屏幕上，之后再输出原文件

    sed的选项

    • 取消默认输出:-n

      取消默认输出，只将文件处理结果打印到屏幕上

    • 改变输出的流向：-i

      将默认输出和处理结果输出到原文件中，而不是屏幕上

      sed -ni 	# 两者组合，可以实现直接改造到原文件，而不在屏幕上输出。

    sed的规则

    ​ 规则写在单引号 ' ' 中

    ​ sed的规则可以概括为 '定位+命令'

    以下是常用命令

    • p和d

      p是将读取到的内容打印输出到屏幕上；d是删除读取到的内容

    • i和a

      i和a都是插入内容，不同的是，i是插入到读取内容之前一行，a是插入到读取内容之后一行。如'i xxxx'

    • s

      's/文本1/文本2/'，表示把文本1替换为文本2
    后可加 g 表示全局替换，而不是只替换每行的第一个
    后可加 i 表示不区分大小写

    常用的定位有两种方式：行号定位和正则定位。

    • 行号定位

      行号组合命令： '行号命令'
比如'1，3d'，删除从第一行到第三行的内容
比如'3i xxxx'、'3a xxxx'，将xxxx添加到第三行之前或第三行之后
比如's/^i/I/'，表示把所有行首的 i 换成 I

    • 正则定位

      正则表达式组合命令： '/正则表达式/命令'

    awk

    文本处理三剑客——awk

    基本用法：

    awk [-options] '规则' [文件]

    awk擅长处理有规律的文本，主要用于格式化处理

    awk的选项

    • 指定行分隔符：-F

      -F 后跟一个字符，表示指定某个字符为行分隔符，将每行数据分为若干段
比如：-F:  表示以 : 为分隔符，如果不加分隔符，默认以空格为分隔符

    awk的规则

    ​ 规则写在单引号 ' ' 中，命令写在花括号 { } 中

    ​ 与sed相同，awk的规则可以概括为'定位+命令'

    以下是常用命令

    • print

      打印输出到屏幕上
比如：{print $1,$3} 把第一段和第三段打印到屏幕上

    awk常用的定位方式也有两种：行号定位和正则定位。

    • 行号定位

      NR == 数字
比如：awk -F: 'NR<=3 || NR=8{print $1,$3}' /etc/passwd  
把该文件的前三行和第八行用 : 分段，打印出第一段和第三段	# $0表示一整行，$NF表示最后一段，$NF-1表示倒数第二段

    • 正则定位

      正则表达式组合命令： '/正则表达式/{命令}'
高阶组合命令：'段号+/正则/+{命令}'		# 表示仅对每行的某一段进行正则匹配
比如：awk -F: '$3 ~ /xxx/{print $0}'	# 打印出第三段是xxx的行，若`~`前加上`!`则表示取反，打印出第三段不是xxx的行

    grep

    文本处理三剑客——grep

    基本用法：

    grep [-options] '过滤规则' [文件]

    grep擅长过滤

    选项

    • 加上行号：-n

      在过滤内容之前显示行号

    • 忽略大小写：-i

      加上-i之后，筛选规则忽略大小写

    • 判断文件是否满足筛选条件：-l

      文件中包含满足筛选条件的行，会输出文件名，否则不输出。

    • 递归：-r

      用于目录，筛选文件夹中的所有文件。常用-l联用：-rl

    • 取反：-v

      满足筛选规则的行不显示，其他的都显示

    规则

    grep的规则本质上是正则表达式

    其他

    • 查看系统进程信息+过滤进程名字
    ps -aux |grep 'vim'			# 过滤出vim相关进程
ps -aux |grep '[v]im'		# 排除掉过滤进程本身
    • 杀掉进程
    kill -9 进程pid				# -9表示强制执行

    Windows系统下找到进程并终止的命令：

    tasklist | findstr "WeChat"			# 利用管道符，找到进程名字为"WeChat"的进程
taskkill /F /PID 12080				# /F 表示强制执行

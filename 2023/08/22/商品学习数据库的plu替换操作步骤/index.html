商品学习数据库的plu替换操作步骤

  1. 数据库文件属性，将只读属性取消掉。
  1. 使用Navicat打开数据库

​ 左上角连接，在选项中找到SQLite，选择现有的数据库新建连接。

  1. 新建一张数据表plu。

​ 两个字段分别保存旧plu和新plu，分别命名为old和new，数据类型为text。

  1. 将plu对应关系导入数据表中
  1. 运行SQL语句

  • Windows

    update feature_code 
set code = (select new from plu where plu.old =feature_code.code)
where feature_code.code in (select old from plu)

  • Android

    update EExportProductSelfLearn
set  productId = (select new from plu where plu.old =EExportProductSelfLearn.productId)
where EExportProductSelfLearn.productId in (select old from plu)
  1. 删掉数据表plu，保存数据库

vim编辑器

vim编辑器分为三种模式：一般模式，编辑模式和命令模式。

使用vim打开一个文本文件时，进入一般模式。

一般模式下输入i,a,o,r等进入编辑模式，编辑模式按esc回到一般模式

一般模式下输入：进入命令模式，命令模式按esc回到一般模式

一般模式

移动光标的方法

  • h j k l，上下左右键

    h: 向左移动一个字符
j: 向下移动一个字符
k: 向上移动一个字符
l: 向右移动一个字符

    hjkl都可以搭配数字来使用，比如8h就是向左移动8个字符，同样方向键也可以。

  • 0和$,end和home

    0: 光标移动到此行的第一个字符之前，相当于home
$: 光标移动到此行的最后一个字符之后，相当于end

  • H M L

    H: 光标移动到屏幕第一行的第一个字符处
M: 光标移动到屏幕中间一行的第一个字符处
L: 光标移动到屏幕最下方一行的第一个字符处

  • 简单的数字

    [数字] + enter : 光标向下移动指定的行数

  • g和G

    G: 光标移动到文件的最后一行
[数字]G : 光标移动到文件的指定行，在set nu模式下很方便
gg: 光标移动到文件的第一行，相当于1G

  • 翻页

    pageup: 向上翻一页
pagedown: 向下翻一页
ctrl + u : 向上翻半页
ctrl + d : 向下翻半页

查找与替换

  • 用/来查找文本

    /字符串 ： 在文件中查找相关字符串，n找下一个，N找上一个

  • 用s和g来替换文本

    :1,$ s/文本1/文本2/g	# 1和$指定行号范围，s表示查找，g表示替换

删除、复制和粘贴

  • 删除字符：x

    x: 删除光标所在字符，相当于delete
[数字]x: 从光标所在处，连续向后删除指定数量的字符

  • 删除行：dd

    dd: 删除光标所在的那一行
[数字]dd: 从光标所在处，向下删除指定数量的行

  • 复制行：yy

    yy: 复制光标所在的那一行
[数字]yy: 从光标所在处，向下复制指定数量的行

  • 粘贴：p和P

    p: 将复制的内容粘贴到光标下一行
P: 将复制的内容粘贴到光标上一行

撤销和重做

  • u和ctrl + r

    u: 撤销上一步操作
ctrl + r : 撤销上一步的撤销

  • .

    . : 这是一个点，代表重复前一个动作

编辑模式

一般模式进入编辑模式的几种方式

  • i和I

    i: 在光标所在处插入
I: 在光标所在行的第一个字符前插入

  • a和A

    a: 在光标所在处的下一个字符插入
A: 在光标所在行的最后一个字符后插入

  • o和O

    o: 在光标所在行的下一行插入
O: 在光标所在行的上一行插入

  • r和R

    r: 取代光标所在字符一次
R: 一直向后移动，取代光标所在文字

命令模式

  • 保存和退出

    w: 保存
w!: 强制保存
q: 退出
q!: 强制退出
wq: 保存并退出
wq!: 强制保存并退出

  • 环境变更

    set nu 显示行号
set nonu 取消显示行号

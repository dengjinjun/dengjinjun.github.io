Jinja语言教程
  1. 语法
    1. 变量
    2. 控制结构

Jinja是一种基于 Python 的模板语言。

  • 官方文档
  • 由 Flask 的作者开发，最初是模仿 Django 的 DTL 模板语言，处理 HTML 文件，后来推广到处理各种类型的文本文件。

语法

  • 三种定界符：

    {{ }}    # 用于读取变量的值，并嵌入模板文本中
{% %}    # 用于执行指令
    # 用于声明单行注释

    渲染模板文件时会删除所有注释。

    声明多行注释的方法：

    {% comment %}
  This is a comment.
  {% endcomment %}

变量

  • 简单变量（如果读取的变量不存在，则取值为空字符串。）

    {% set a='Hello' %}   # 创建变量
{% set b=a %}         # 将变量赋值给另一个变量
{{ b }}               # 读取变量的值

  • 变量可以是 Python 中任意类型的对象，可以调用该对象的属性、方法：

    Explain{{ name }}            # 读取变量
{{ name.__doc__ }}    # 调用属性
{{ name.encode() }}   # 调用方法
{{ dict1['A'] }}      # 读取字典
{{ list1[0] }}        # 读取列表

控制结构

  • if 语句：

    Explain{% if a is defined %}   # 判断变量是否已定义
  AAA
  {% elif not b %}
  BBB
  {% else %}
  CCC
  {% endif %}

    {{ True if a else False }}
{{ a or 'Hello' }}      # 为变量设置默认值

  • 迭代列表：

    Explain<ul>
  {% for i in list1 %}
    
  • {{ i }}
    • 
    {% endfor %}
</ul>

  • 迭代字典：

    Explain<dl>
  {% for k, v in dict1.items() %}
    
    {{ k }}
    
    
    {{ v }}
    
    {% endfor %}
</dl>

  • 继承：

    Explain{% extends "base.html" %}   # 继承 base.html 的内容

{% block head %}            # 定义一个 block 块，如果继承的父文件中存在同名的 block ，则会覆盖它
      {{ super() }}           # 插入父文件中同名的 block 的内容
      Hello
  {% endblock %}

面向对象三大特性

  • 封装

    • 将数据打包放在对象中

    • 将同一类型的函数归类到一个类中

  • 继承

    提高代码的重用性，子类可以使用父类中的方法。多个类可以把公共的功能提取到父类中

  • 多态

    多态指多种形态，多种类型，python中默认支持多态

数组（array）是一个包含多个值的变量。成员的编号从0开始，数量没有上限，也没有要求成员被连续索引。

创建数组

逐个赋值

数组可以采用逐个赋值的方法创建，数组名[索引]=元素值，注意数组第一个元素的索引是0， 而不是1。

ARRAY_NAME[index]=value

一次性赋值

数组也可以采用一次性赋值的方式创建。

# 按照默认顺序赋值，不同值用空格分开
ARRAY_NAME=(value1 value2 ... )

# 也可以在每个值前面指定位置
ARRAY_NAME=([2]=c [0]=a [1]=b)

# 可以的只为某些值指定位置，`hatter`是数组的0号位置，`duchess`是5号位置，`alice`是6号位置，没有赋值的数组元素的默认值是空字符串
ARRAY_NAME=(hatter [5]=duchess alice)

# 定义数组的时候，可以使用通配符，比如将当前目录的所有 MP3 文件，放进一个数组
$ mp3s=( *.mp3 )

事先声明

declare -a命令可以事先声明一个数组

$ declare -a ARRAYNAME

read -a命令则是将用户的命令行输入，存入一个数组。

$ read -a dice

读取数组元素

读取单个元素

读取数组指定位置的成员，要使用下面的语法。

$ echo ${array[i]}     # i 是索引，语法里面的大括号是必不可少的

注意：给数组赋值的时候以及读取数组成员的时候，如果不指定索引，默认是读取和引用数组的首个元素foo[0]

读取所有元素

@和*是数组的特殊索引，表示返回数组的所有成员。

$ foo=(a b c d e f)
$ echo ${foo[@]}
a b c d e f

这两个特殊索引配合for循环，就可以用来遍历数组。

for i in "${names[@]}"; do
  echo $i
done

注意：@和*放不放在双引号之中，是有差别的，为了能够正确地显示结果，建议放在双引号之中。

拷贝数组

拷贝一个数组的最方便方法，就是写成下面这样。

$ hobbies=( "${activities[@]}" )

这种写法也可以用来为新数组添加成员。

$ hobbies=( "${activities[@]}" diving )

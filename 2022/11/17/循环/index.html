Bash 提供三种循环语法for、while和until。

while 循环

while循环有一个判断条件，只要符合条件，就不断循环执行指定的语句。

while condition
do
  commands
done

注意：

  1. 关键字do和while写在同一行时，需要分号分隔。写成两行时不需要分号。
  2. 除了多行的写法，while结构也可以写成单行：while true; do echo 'Hi, while looping ...'; done
  3. while的条件部分可以是执行一个命令，根据命令的返回值作为判断条件。
  4. while的条件部分也可以执行任意数量的命令，但是执行结果的真伪只看最后一个命令的执行结果。
  5. 循环条件condition可以使用test命令，跟if结构的判断条件写法一致。

until 循环

until循环与while循环恰好相反，只要判断条件为失败，就循环执行指定的语句，直到条件成立（判断为真）。

until condition
do
  commands
done

注意：

  1. 关键字do与until写在同一行时，需要分号分隔。写成两行时不需要分号。
  2. until的条件部分也可以是一个命令，表示在这个命令执行成功之前，不断重复尝试。
  3. 一般来说，until用得比较少，until循环都可以转为while循环，完全可以统一都使用while。比如下面的例子。

until循环转为while循环：

until cp $1 $2; do
 echo 'Attempt to copy failed. waiting...'
 sleep 5
done

while ! cp $1 $2; do
 echo 'Attempt to copy failed. waiting...'
 sleep 5
done

for…in 循环

for...in循环用于遍历列表的每一项。

for variable in list
do
  commands
done

for循环会依次从list列表中取出一项，作为变量variable，然后在循环体中进行处理。

注意：

  1. 关键字for与do写在同一行时，需要分号分隔。写成两行时不需要分号。

  2. list有多种形式，以下是一些例子

    • for i in word1 word2 word3;		# 给出多个值作为列表

    • for i in *.png; 		# 由通配符产生，查看当前目录下的所有png文件

    • for i in $(cat ~/.bash_profile);		# 由子命令产生

    • for filename in "$@" ;		# 读取脚本的所有参数，可以直接简写为`for filename;`，但是可读性不好。

    • for in in {1..10..2};		# 使用{start..end..step}扩展语法生成一个序列

for 循环

for循环还支持 C 语言的循环语法。

for (( expression1; expression2; expression3 ))
do
  commands
done

expression1用来初始化循环条件，expression2用来决定循环结束的条件，expression3在每次循环迭代的末尾执行，用于更新值。

它等同于下面的while循环：

(( expression1 ))
while (( expression2 )); do
  commands
  (( expression3 ))
done

注意：

  1. 循环条件放在双重圆括号之中。

  2. 圆括号之中使用变量，不必加上美元符号$。

  3. for条件部分的三个语句，都可以省略。比如下面的例子：

    # 脚本会反复读取命令行输入，直到用户输入了一个点（`.`）为止，才会跳出循环。
for ((;;))
do
  read var
  if [ "$var" = "." ]; then
    break
  fi
done

break和continue

Bash 提供了两个内部命令break和continue，用来在循环内部跳出循环。

break

break命令立即终止循环，程序继续执行循环块之后的语句，即不再执行剩下的循环。

#!/bin/bash

# 打印3行结果。一旦变量`$number`等于3，就会跳出循环，不再继续执行。
for number in 1 2 3 4 5 6
do
  echo "number is $number"
  if [ "$number" = "3" ]; then
    break
  fi
done

continue

continue命令立即终止本轮循环，开始执行下一轮循环。

#!/bin/bash

# 只要用户输入的文件不存在，`continue`命令就会生效，直接进入下一轮循环（让用户重新输入文件名）
while read -p "What file do you want to test?" filename
do
  if [ ! -e "$filename" ]; then
    echo "The file does not exist."
    continue
  fi

  echo "You entered a valid file.."
done

select 结构

select结构主要用来生成简单的菜单。它的语法与for...in循环基本一致。

select生成一个菜单，内容是列表list的每一项，并且每一项前面还有一个数字编号。Bash 提示用户选择一项，输入它的编号。用户输入以后，Bash 会将该项的内容存在变量name，该项的编号存入环境变量REPLY，执行命令体commands。如果用户没有输入，就按回车键，Bash 会重新输出菜单，让用户选择。执行结束后，回到第一步，重复这个过程。

select name in list
do
  commands
done

select可以与case结合，针对不同项，执行不同的命令。下面是一个例子：

#!/bin/bash

echo "Which Operating System do you like?"

select os in Ubuntu LinuxMint Windows8 Windows10 WindowsXP
do
  case $os in
    "Ubuntu"|"LinuxMint")
      echo "I also use $os."
    ;;
    "Windows8" | "Windows10" | "WindowsXP")
      echo "Why don't you try Linux?"
    ;;
    *)
      echo "Invalid entry."
      break
    ;;
  esac
done
